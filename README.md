# Dotfiles
The purpose of this repository is simply to keep track of my dotfiles, feel free to use anything that you find useful here :D

## System Information
Some context on what software is running on my system

    Desktop Enviroment:
        hyprland    / Window Manager
        fish        / Command Line Interpreter
        wofi        / App Launcher
        kitty       / Terminal emulator
        waybar      / Status bar
        swaync      / Notifications
        hyprlock    / Lockscreen
    Programs:        
        nvim        / Text editor
        yazi        / File Manager
        btop        / Task Manager
        fastfetch   / Terminal system overview
