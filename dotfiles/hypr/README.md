# Hyprland
Configuration for the full Hyprland compositor stack.

## What's configured
- `hyprland` — window manager rules, keybinds and animations
- `hyprpaper` — wallpaper settings
- `hyprlock` — lock screen layout and styling
- `hyprshot` — screenshot keybinds

## Dependencies
- `hyprland` `hyprpaper` `hyprlock` `hyprshot`
